# /notion-search

Notion知識ベース全体からキーワード検索するスキル。

## 使い方

```
/notion-search "検索キーワード"
```

## 動作

1. 知識ベース (33d93e3d-9d44-8037-85a5-000bc9d1f570) をキーワードで検索
2. マッチしたページをリストアップ
3. 各ページのタイトル・URL・簡潔な説明を表示

## 例

```
/notion-search "API設計"
/notion-search "パフォーマンス最適化"
/notion-search "セキュリティ"
```

検索結果から必要なページが見つかったら、 `/notion-read-page` で詳細を読み込んでください。

## 対象Database

- **Knowledge Base** (知識ベース): 仕様書・設計ドキュメント・過去の決定事項

## 制限

- 読み取り専用（検索のみ）
- 検索は知識ベースのみ対象
